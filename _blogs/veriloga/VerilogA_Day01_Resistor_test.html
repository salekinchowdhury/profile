VerilogA Day 01: Resistor

Posted on: October 24, 2024

In this blog post, we will cover how to model a simple resistor using Verilog-A. Resistors are fundamental components in analog circuits, and understanding how to model them in Verilog-A is important for circuit designers.

Resistor Model in Verilog-A

Below is an example of the Verilog-A code for a simple resistor model:


  module resistor(p, n);
     electrical p, n;
     parameter real r = 1k;
     analog V(p, n) <+ r * I(p, n);
  endmodule
